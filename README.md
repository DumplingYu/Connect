# Connect

A Python-based chat room software.

Latest Version: Server: **Alpha 2.0**  Client: **Alpha 2.1**

Python Version: **3.5**

Supported OS: **OSX / macOS 10.9+**

##Features
* GUI
* Push and sound notification
* User nicknames
* Custom skins
